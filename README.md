## Overview
This project documents the design and implementation of a small Security Operations Center (SOC) home lab.
The goal is to practice log collection, detection engineering, and incident response using realistic telemetry.

## Architecture
- Windows endpoint with Sysmon enabled
- Ubuntu-based Wazuh manager and dashboard
- Centralized log collection and analysis

## Data Sources
- Windows Security Event Logs
- Sysmon operational logs

## Objectives
- Generate high-fidelity endpoint telemetry
- Centralize and analyze logs
- Create detections for suspicious activity
- Document investigations using incident response methodology

Initial repository structure created. Lab build in progress.
